# Project Manager App
Ongoing development of a project management app for an undergraduate thesis.

# Technologies
## Backend
- Java
- Spring Boot
- JPA / Hibernate
- Maven
- H2

## Frontend
- HTML5
- CSS3
- JS / TypeScript
- Angular

# How to run

## clone repository
git clone https://github.com/MarcosAndreBueno/project-manager

## Backend
Requirements:
- Java 25

```bash
cd backend
./mvnw spring-boot:run
```

## Frontend
Requirements:
- node 24 
- npm 11
- angular cli 21

```bash
cd frontend
npm install --save-dev @angular-devkit/build-angular
npm run start
```
